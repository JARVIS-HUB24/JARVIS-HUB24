### About me

I'm a backend developer focused on the intersection of software engineering and intelligent systems.
My main interest is designing and building **autonomous agents** — systems that can reason, plan, and act with minimal human input.

I track developments across the AI research landscape closely: from foundation models and multi-agent frameworks to inference optimization and tool-use architectures. When something looks promising, I don't just read about it — I build with it.

- **Building:** automation tools, agent infrastructure, data pipelines
- **Researching:** autonomous AI systems, agent orchestration, LLM integrations
- **Approach:** deep analysis first, practical implementation second — no cargo-culting
- **Currently exploring:** agentic workflows, self-improving systems, multi-model pipelines
